# Гусь & Лебедь — система доставки еды

Веб-приложение для заказа еды с доставкой. Поддерживает роли: клиент, оператор, курьер, администратор.

## Быстрый старт

```bash
npm install
npm run seed    # заполнить БД тестовыми данными (один раз)
npm start       # запуск сервера на http://localhost:3000
```

## Настройка

Скопируйте `.env.example` в `.env` и измените секрет:

```
JWT_SECRET=ваш-секретный-ключ
PORT=3000
```

## Тестовые аккаунты

| Роль      | Телефон (логин) | Пароль       |
|-----------|-----------------|--------------|
| Админ     | admin           | admin123     |
| Оператор  | operator        | operator123  |
| Клиент    | client          | client123    |
| Курьер    | courier         | courier123   |

## Роли и возможности

- **Клиент** — просмотр меню, корзина, оформление заказов, история заказов
- **Оператор** — управление заказами, назначение курьера, отмена заказов
- **Курьер** — активные доставки, подтверждение доставки, статистика заработка
- **Админ** — dashboard, управление меню и категориями, сотрудники

## API (основные эндпоинты)

| Метод | Путь | Описание |
|-------|------|----------|
| POST | /api/register | Регистрация (только client) |
| POST | /api/login | Вход |
| GET | /api/menu | Меню (публичное) |
| GET | /api/orders/:id | Детали заказа с составом |
| PATCH | /api/menu/:id | Редактирование блюда (admin) |
| PATCH | /api/orders/:id/status | Смена статуса (operator/admin) |

## Тестирование
